\begin{abstract}
Wave inverse problems require recovering local structure from measurements
formed by global propagation physics. We introduce SRU-Net, a spectral-residual
U-Net for physics-constrained wave reconstruction that combines Fourier-style
global mixing with convolutional local spatial processing. On coherent diffractive imaging
with synthetically-generated diffraction data, SRU-Net gives the strongest
reconstruction results among matched convolutional neural network (CNN),
Fourier neural operator (FNO), and factorized Fourier neural operator (FFNO)
comparisons. We also evaluate the same architecture on PDEBench compressible
Navier-Stokes as a secondary forward partial differential equation (PDE)
benchmark. In that setting SRU-Net is competitive with neural-operator
baselines and remains ahead of FNO and U-Net rows, while FFNO is the
lowest-error model. These results support spectral-local residual networks for wave inverse
reconstruction, with PDE prediction serving as a transfer test rather than the
primary result.
\end{abstract}

\section{Introduction}

Wave propagation inverse problems recover structure from measurements produced
by a known physical propagation process. Coherent diffractive imaging (CDI) is a
representative example: detectors measure far-field diffraction intensities
rather than object phase, so reconstruction must infer a complex object from
intensity-only wave measurements. CDI is therefore a phase-retrieval problem:
the missing phase of the diffracted wave must be inferred from intensity
measurements and physical constraints. Classical iterative methods can recover
high-quality reconstructions, but they are often too slow for high-throughput
microscopy, \emph{in situ} experiments, and streaming beamline operation.

Physics-constrained deep reconstruction addresses this bottleneck by combining
the speed of neural inference with a differentiable physical forward model. In
PtychoPINN, Hoidn, Mishra, and Mehta showed that an unsupervised CDI network
can be trained by mapping predicted objects through real-space overlap
constraints and far-field diffraction, then minimizing measurement-space error
under photon-counting statistics \cite{hoidn2023physics}. This establishes the
training setup used for our primary wave inverse benchmark: the network is not
trained only to match labels, but is constrained to produce reconstructions
whose predicted diffraction agrees with the measurements.

The remaining question is architectural. The inverse map from diffraction
measurements to object structure is neither a pure image-to-image translation
problem nor a standard forward operator-learning problem. Diffraction globally
couples object regions through Fourier-domain propagation, while reconstruction
quality is determined by local spatial features such as edges, sparse lines,
and phase structure.

SRU-Net instantiates these requirements with spectral-local encoder blocks, a
residual bottleneck, and a multiscale convolutional decoder. The spectral path
provides global coupling, the local convolutional path preserves local spatial
features, the bottleneck performs residual refinement, and the decoder maps the
mixed representation back to a spatially precise reconstruction. CNN and U-Net
baselines emphasize local spatial priors but lack explicit spectral global
mixing. FNO, FFNO, and U-NO provide strong neural-operator baselines, but can be
limited when the inverse task requires localized image synthesis from globally
coupled measurements. The experiments test this design by comparing SRU-Net
with local and neural-operator baselines and by varying where spectral
components appear in the architecture.

The primary evaluation is a 128-by-128 CDI benchmark with
synthetically-generated diffraction data, where SRU-Net is compared with CNN,
FNO, and FFNO reconstruction models under a matched physics-consistency
protocol. Here and below, ``+ PINN'' denotes a physics-informed neural network
training objective. In this work the PINN objective is the CDI
physics-consistency loss: the network prediction is mapped through the known
overlap-consistency and diffraction operators, and the loss is applied in
measurement space rather than directly against paired object targets. We also
evaluate the same architecture on
PDEBench 2D compressible Navier-Stokes (CNS) as a secondary forward-PDE
benchmark. CNS tests whether the global-local inductive bias transfers to a
different physical field-mapping task. In that benchmark, FFNO remains the best
row, while SRU-Net is competitive and improves over FNO and U-Net baselines.

This paper contributes a spectral-local residual network for wave inverse
reconstruction, a shared mathematical description of the architecture across CDI
reconstruction and CNS forward prediction, and a two-part experimental program
that makes a primary claim on CDI with synthetically-generated diffraction data
while using PDEBench CNS as secondary transfer evidence.

\section{Related Work}

\subsection{Operator learning for wave and PDE field maps}

Neural operators learn mappings between fields and have become a central model
class for scientific machine learning. Fourier neural operators (FNOs) apply
global spectral convolutions to represent solution operators for parametric PDEs
\cite{fno}. Factorized Fourier neural operators (FFNOs) make this
parameterization substantially deeper and more efficient by factorizing Fourier
transforms, sharing the kernel integral operator across layers, and using
residual temporal updates \cite{ffno}. This combination gives FFNO reported
state-of-the-art-class performance on PDE benchmarks, including
Navier-Stokes-style settings, which makes it a strong baseline rather than only
an efficient FNO variant. The global receptive field of these models is
attractive for wave propagation, diffraction, and fluid dynamics, where distant
regions of a field are coupled by the governing physics. PDEBench provides a
controlled setting for evaluating such models on forward physical prediction
tasks, including compressible Navier-Stokes dynamics \cite{pdebench}.

The same global-coupling argument motivates operator-style models for wave
inverse problems. In diffraction imaging, local object structure is observed
through global Fourier measurements; in compressible flow, local shocks and
vortices interact with coherent large-scale dynamics. Pure spectral models can
represent global coupling directly, but Fourier-mode truncation can bias
predictions toward lower-frequency structure. The central architectural problem
is therefore not only how to model field-wide interactions, but how to retain
localized detail while doing so.

\subsection{Physics-constrained wave inverse reconstruction}

Hoidn, Mishra, and Mehta introduced PtychoPINN for unsupervised scanning-CDI
reconstruction \cite{hoidn2023physics}. The method learns an inverse map from
diffraction data to a complex real-space object, then trains it only after
passing the predicted object through known coherent diffractive imaging physics:
overlap-based
real-space consistency, the probe interaction, and far-field diffraction. Its
loss is defined in photon-count space with a Poisson model, so the network is
optimized against the measured diffraction statistics rather than paired
ground-truth objects.

This paper uses that physics-constrained formulation as the primary evaluation
setting for a broader wave inverse architecture. PtychoPINN supplies the
measurement-consistency objective for CDI, but the model contribution is not a
new coherent diffractive imaging forward model. The question is whether the reconstruction
network used inside a wave inverse problem should remain a local CNN-style image
model, or whether it should combine local spatial processing with explicit spectral
mixing.

\subsection{Spectral-local architectures}

Several recent architectures combine spectral mixing with local or multiscale
image structure. U-FNO-style models place Fourier layers within a U-Net-like
multiresolution architecture \cite{ufno}, while localized-kernel neural
operators strengthen the local component of operator models \cite{localno}.
These approaches reflect a common design principle: spectral layers are useful
for field-wide coupling, but local convolutional and residual paths remain
important for high-frequency detail and spatially localized structure.

SRU-Net follows this principle in a network designed first for wave inverse
reconstruction. It combines spectral-local encoder blocks, residual refinement,
and decoder-side upsampling as a single reconstruction architecture trained
through a physics-consistency loss. The PDEBench CNS experiments then test the
same architecture in a supervised forward-prediction setting, where FNO and FFNO
are natural operator-learning baselines and where SRU-Net should be interpreted
as competitive transfer evidence rather than as the central benchmark result.

\section{Methods}

\subsection{Task-agnostic network formulation}

Let $x$ denote the task input and $y$ the target field or reconstruction. We
write the learned network as
\begin{equation}
  \hat{y} = G_{\theta}(x).
\end{equation}
For CDI, $x$ is a stack of diffraction measurements and $\hat{y}$ is a complex
object-patch representation, meaning local crops of the reconstructed complex
object represented by real and imaginary channels. For PDEBench CNS, $x$ is a
temporal history of physical fields and $\hat{y}$ is the predicted next state. The tasks use
different input representations and objective functions, but share the same
architectural principle:
$G_{\theta}$ combines global spectral mixing with local convolutional feature
recovery.

We decompose the network as
\begin{equation}
  G_{\theta} = D_{\theta} \circ B_{\theta} \circ E_{\theta},
\end{equation}
where $E_{\theta}$ is the spectral-local encoder, $B_{\theta}$ is a ResNet
bottleneck at reduced spatial resolution, and $D_{\theta}$ is a convolutional
upsampling decoder.

We use subscripts to distinguish architectural positions rather than tensor
dimensions. The index $\ell$ denotes the encoder block that maps
$z_{\ell}$ to $z_{\ell+1}$; $j$ indexes the six residual bottleneck blocks;
$s$ indexes resolution levels used for encoder-decoder skip connections; and
$k$ indexes Fourier modes. The retained Fourier-mode set is $K$, with
$m=|K|$ in the architecture diagram. Superscripts such as $B_j^{\times 6}$
denote repeated blocks rather than exponentiation. Task subscripts such as
${\rm CDI}$ and ${\rm CNS}$ label task-specific outputs.

\subsection{Spectral-local encoding}

For a feature tensor $z_{\ell}$, a spectral-local encoder block has the form
\begin{equation}
z_{\ell+1}
= \sigma\left(
L_{\ell} z_{\ell}
+ \mathcal{F}^{-1}
\left[
R_{\ell}(k)\,\mathcal{T}_{K}\left(\mathcal{F}(z_{\ell})\right)(k)
\right]
\right),
\end{equation}
where $L_{\ell}$ is a learned local convolutional map, $\mathcal{F}$ is the 2D
Fourier transform, $\mathcal{T}_{K}$ retains a fixed set of Fourier modes,
$R_{\ell}(k)$ is a learned complex spectral mixing tensor over retained modes,
and $\sigma$ is the GELU nonlinearity used in the implemented SRU-Net. In
residual form, this can be written as
\begin{equation}
z_{\ell+1}
= z_{\ell} + \sigma\left(L_{\ell} z_{\ell} + S_{\ell} z_{\ell}\right),
\end{equation}
with $S_{\ell}$ denoting the truncated spectral operator. The local path
preserves high-frequency and edge-like information that may be weakened by
spectral truncation, while the spectral path provides global coupling.

\subsection{Spectral-convolutional residual architecture}

The implemented SRU-Net consists of an initial convolution, four spectral-local
encoder blocks, two downsampling stages, a
six-block residual bottleneck at $N/4$ spatial resolution, and a convolutional
upsampling decoder. Figure~\ref{fig:srunet_architecture} summarizes the data flow, and
Table~\ref{tab:architecture_summary} lists the architectural choices that
determine the shape of the model.

In the architecture diagram, $\phi_0$ is the initial channel projection, $E_0$
through $E_3$ are encoder blocks, $B_j$ denotes a residual bottleneck block, and
$U_1,U_2$ are decoder upsampling stages. The spatial resolution is denoted by
$N$, $C_{\rm in}$ is the task-specific input channel count, $m$ is the number of
retained Fourier modes, and $e_s,d_s$ are encoder and decoder features at scale
$s$. The skip projection $P_s$ maps encoder features into the decoder channel
space before additive fusion.
Within encoder block $E_\ell$, $z_\ell$ is the input feature tensor,
$S_\ell=\mathcal F^{-1}R_\ell\mathcal T_K\mathcal F$ is the truncated spectral
operator, $L_\ell$ is the local convolutional path, and $\sigma$ is GELU. Here
$\mathcal F$ is the Fourier transform, $\mathcal T_K$ keeps the retained modes,
and $R_\ell$ mixes those modes. Within bottleneck block
$B_j$, $u_j$ is the bottleneck feature tensor,
$R_j=\ConvTT\cdot\mathrm{InstNorm}\cdot\mathrm{GELU}
\cdot\ConvTT\cdot\mathrm{InstNorm}$ is applied before the
residual addition, and $\alpha_j$ is its learned residual scale; the centered
dot lists operations in left-to-right order. Here $\ConvTT$ denotes a
3-by-3 convolution. Within decoder stage $D_s$, $U_s$ upsamples the coarser
decoder state and $P_s$ projects the matching encoder feature before fusion.

When skip connections are enabled, encoder features $e_s$ are fused into decoder
states $d_s$. For additive fusion,
\begin{equation}
  d_s = U_s(d_{s+1}) + P_s(e_s),
\end{equation}
where $U_s$ upsamples the coarser decoder state and $P_s$ projects the encoder
feature into the decoder channel space. For concatenative fusion,
\begin{equation}
  d_s = P_s\left([U_s(d_{s+1}), e_s]\right).
\end{equation}
The CDI network outputs complex object patches in real/imaginary format. In the
PDEBench experiments, the same architecture is evaluated on real-valued physical
field histories and next-state targets rather than diffraction measurements.

\subsection{Physics-consistency loss for CDI}

For CDI, SRU-Net predicts local complex object patches,
\begin{equation}
  \hat{o} = G_{\theta}(x).
\end{equation}
The prediction is passed through a real-space consistency operator $F_c$, which
stitches overlapping local patches into a common object canvas and re-extracts
position-consistent patches. The consistent object patches are then passed
through a known diffraction map $F_d$:
\begin{equation}
  \hat{x}
  = F_d\left(F_c(\hat{o}; r); P\right),
\end{equation}
where $r$ denotes scan positions and $P$ is the probe. The CDI objective is
defined in measurement space:
\begin{equation}
  \mathcal{L}_{\mathrm{CDI}}
  = \mathcal{D}\left(x, F_d(F_c(G_{\theta}(x); r); P)\right)
  + \lambda R(G_{\theta}(x)).
\end{equation}
Here $\mathcal{D}$ is the experiment-specific measurement loss, such as Poisson
negative log likelihood or amplitude/intensity reconstruction error, and $R$
denotes optional reconstruction regularization. This preserves the
physics-informed training principle: the network is optimized through the
forward diffraction model rather than only through a supervised object-target
loss.

\subsection{PDEBench CNS supervised prediction}

For the CNS benchmark, the same architecture is evaluated as a supervised
forward predictor. Let
\begin{equation}
  u_t = [\rho_t, v_{x,t}, v_{y,t}, p_t]
\end{equation}
be the density, velocity, and pressure state. Given a history length $h$, the
input is
\begin{equation}
  x_t = [u_{t-h}, \ldots, u_{t-1}],
\end{equation}
and the model predicts
\begin{equation}
  \hat{u}_t = G_{\theta}(x_t).
\end{equation}
The training objective is mean-squared error on normalized physical fields:
\begin{equation}
  \mathcal{L}_{\mathrm{CNS}}
  = \left\|\hat{u}_t - u_t\right\|_2^2.
\end{equation}
Evaluation is performed after denormalization and reports root mean squared
error (RMSE), normalized RMSE (nRMSE), relative $L_2$, and Fourier-band RMSE
over low, mid, and high radial-frequency bands. The
high-frequency metric is included because shocks and localized vortical features
are precisely where global spectral models may lose detail.

\section{Experiments}

\subsection{Primary CDI benchmark}

The primary benchmark is a 128-by-128 CDI reconstruction task with
synthetically-generated diffraction data. Each sample is generated from sharp
line-like objects, a known probe, and simulated far-field diffraction
measurements. This benchmark is the direct test of the physics-informed
reconstruction model: the network predicts complex object patches, the CDI
consistency and diffraction operators map the prediction back to detector space,
and reconstruction quality is measured against held-out object amplitude and
phase. We refer to this benchmark as Lines128.

The Lines128 comparison uses $N=128$ object grids, two training objects, two
held-out test objects, $10^9$ photons per exposure, 40 training epochs,
learning rate $2\times 10^{-4}$, ReduceLROnPlateau scheduling,
and complex-valued real/imaginary network output.

CDI results are reported by model and training procedure: SRU-Net + PINN,
FFNO + PINN, FNO + PINN, and CNN + PINN denote architectures trained through
the CDI diffraction forward model rather than through direct
supervised object targets. Rows without ``+ PINN'' denote non-PINN baselines.

\subsection{Secondary cross-domain benchmark: PDEBench CNS}

The secondary benchmark is PDEBench 2D compressible Navier-Stokes at native
$128\times128$ resolution. It stresses the same architectural tradeoff in a
different physical setting: coherent global flow and wave structure must be
predicted while preserving localized shocks, vortices, and high-gradient field
features. It therefore tests whether the SRU-Net architecture
transfers beyond the CDI inverse problem.

The selected CNS data are the official 128-by-128 periodic compressible-flow
training trajectories at Mach number $M=1.0$, shear viscosity
$\eta=0.01$, and bulk viscosity $\zeta=0.01$. The spatial grid has
$\Delta x=\Delta y=1/128=0.0078125$, the temporal spacing is
$\Delta t=0.05$, and the spatial boundary condition is periodic, meaning that
values wrap around at the domain edges as on a torus. Each trajectory contains
the fields density, $v_x$, $v_y$, and pressure on a $128\times128$ grid.
Splits are trajectory-level, normalization is computed from the train split
only, and metrics are evaluated on denormalized predictions.

\paragraph{Spectral-component ablations.}
Table~\ref{tab:cdi_arch_ablations} tests whether SRU-Net's CDI advantage is
explained by simply adding or moving spectral components. Replacing the SRU-Net
bottleneck with an FFNO-style bottleneck worsened both amplitude and phase
error. A lighter linear decoder for the spectral-bottleneck model regressed
sharply in amplitude quality, while reducing the spectral-bottleneck model to
one downsampling stage improved phase MAE but lost amplitude fidelity. Spectral
coupling was most effective when paired with local convolutional paths,
residual refinement, and convolutional upsampling; adding or moving spectral
components alone did not reproduce the SRU-Net result.

\section{Discussion}

The core architectural claim is that wave inverse reconstruction benefits from
networks that encode both propagation-scale coupling and local spatial features.
The task physics should constrain the output through a differentiable forward
model, while the reconstruction network should match the structure of the
inverse map. In CDI, diffraction couples the object through Fourier-domain wave
physics, while the object itself contains sharp local features. In compressible
flow, used here as a secondary transfer setting, large-scale coherent dynamics
coexist with localized shocks and vortices.

The SRU-Net encodes this global-local structure directly. Fourier-style
encoder blocks provide global mixing; local convolutional branches and decoder
features preserve high-frequency information; the ResNet bottleneck gives the
model capacity to refine features at reduced spatial resolution. This differs
from both pure CNN reconstruction and pure spectral operator modeling.

The CDI architecture table and spectral-component ablations point to the same
conclusion: SRU-Net's performance depends on where spectral coupling is
introduced and how it is paired with local convolutional paths, residual
refinement, and decoder-side upsampling. A remaining encoder-branch ablation
will remove one path at a time under the same CDI dataset, training, and
evaluation protocol.

The evidence also argues for a narrow interpretation. The primary claim is about
wave inverse reconstruction: the CDI result compares local CNN, FNO, FFNO,
SRU-Net, and spectral SRU-Net rows under one synthetically-generated
diffraction data protocol,
while the supervised FFNO control isolates one training-procedure contrast. The
CNS results use 512 training trajectories in the family comparison table as
secondary cross-domain architecture evidence. They show that SRU-Net is
competitive in a forward-PDE setting and improves over FNO and U-Net baselines,
but they do not make SRU-Net the best CNS model; FFNO is the lowest-error CNS
row. The SRU-Net$^*$ notation marks the CNS spectral-bottleneck row.
